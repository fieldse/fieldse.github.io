# Static Site Blog

A minimal Jekyll blog deployed on GitHub Pages.

**Live site:** [mattfields.dev](https://mattfields.dev) · [fieldse.github.io](https://fieldse.github.io)

## Tech Stack

- **Jekyll** — Static site generator
- **GitHub Pages** — Hosting
- **Hydejack v9** — Theme
- **Markdown** — Content format

## Adding Posts

```bash
./newpost.sh My Post Title
```

Posts go in `docs/_posts/` as `YYYY-MM-DD-title.md`. Push to publish.
